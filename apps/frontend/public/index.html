About This Application

IWA-Microservices is a deliberately insecure microservices-based pharmacy application designed for application security testing, DevSecOps demonstrations, and security training.

API Gateway: http://localhost:8080
Documentation: See README.md for full API documentation

Microservices

Catalog Service

Product catalog management

http://localhost:8081/api/products

SQL Injection

Customers Service

User authentication and management

http://localhost:8082/api/customers

Weak Auth Plain Text Passwords

Orders Service

Order management

http://localhost:8083/api/orders

Insecure Deserialization IDOR

Payments Service

Payment processing

http://localhost:8084/api/payments/config

Hardcoded Secrets

http://localhost:8085/api/prescriptions

Quick Test Examples

SQL Injection (Catalog)

curl "http://localhost:8081/api/products/search?q=test' OR '1'='1"

Authentication Bypass (Customers)

curl -X POST http://localhost:8082/api/customers/login -H "Content-Type: application/json" -d '{"username":"admin","password":"x' OR '1'='1"}'

IDOR (Orders)

curl http://localhost:8083/api/orders/1
curl http://localhost:8083/api/orders/2

View Hardcoded Secrets (Payments)
